# Jocke Ohlssons Måleri — website (draft)

Single-page landing site for **Jocke Ohlssons Måleri** (Töreboda / Skaraborg) built with **Next.js (App Router)** + **Tailwind CSS v4**.

The website is intentionally a draft: photos and some final customer details can be added later without rewriting components.

## Development

Install dependencies:

```bash
npm install
```

Run the dev server:

```bash
npm run dev
```

Open http://localhost:3000

## Where to edit content

Most copy and business info is centralized here:

- `src/lib/content.ts`

## Adding project images later

1. Put images in:
   - `public/projects/`
2. Add `imageSrc` + `alt` on the matching project item in:
   - `src/lib/content.ts`

If `imageSrc` is missing, the UI keeps showing “Bild kommer” placeholders.

## Contact form (email)

The contact form posts to `/api/contact`.

To enable email sending, copy `.env.example` → `.env.local` and fill in SMTP credentials:

- `SMTP_HOST`, `SMTP_PORT`, `SMTP_USER`, `SMTP_PASS`
- `CONTACT_TO_EMAIL`, `CONTACT_FROM_EMAIL`

If not configured, the UI tells the visitor to call instead (phone CTA).

## SEO / sharing

The site includes `robots.txt`, `sitemap.xml`, and an autogenerated Open Graph image.

When deploying, update the placeholder domain:

- `src/app/layout.tsx` (`metadataBase`)
- `src/app/robots.ts` (`SITE_URL`)
- `src/app/sitemap.ts` (`SITE_URL`)
